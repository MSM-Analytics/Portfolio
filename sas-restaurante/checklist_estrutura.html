Checklist Estrutura Projeto Angular

1️⃣ Layouts

  • ProtectedLayout — sidebar + menu + logout
  • PublicLayout (opcional) — layout de login

2️⃣ Core

  • auth.service.ts — login, logout, token, hasPermission(action: string)
  • auth.guard.ts — protege rotas
  • token.storage.ts — gerenciar token no localStorage
  • has-permission.directive.ts (opcional) — mostrar/esconder elementos por ação

  • /gestao/products — CRUD básico

  • /gestao/tenants — CRUD básico
